## Applying Deep Learning to Petroleum Well Data
This repository contains the source files required to reproduce the results in "Applying Deep Learning to Petroleum Well Data." This README will explain how these files should be used.

### Dependencies
- [Python 2.7](https://www.python.org/)
- [Numpy/Matplotlib](http://www.scipy.org/)
- [Theano](http://deeplearning.net/software/theano/)
- [Keras](http://keras.io/)
